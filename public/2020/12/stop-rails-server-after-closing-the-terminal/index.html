Stop Rails Server After Closing the Terminal

published Sunday, Dec 6, 2020
last edit Sunday, Dec 6, 2020

I always accidentally close the terminal window where my local rails server process is running, and every time I have to look up the command to find the process id and kill it so I can restart the server. So I’m just writing it down here:

lsof -i tcp:3000
kill <PID>
